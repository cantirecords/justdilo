---
name: Notification tone — friend, not robot
description: All push notifications and reminders must feel like a message from a close friend, not a productivity app
type: feedback
---

All push notifications and reminders in JustDilo must feel like they come from a close friend — warm, personal, human. Never robotic, corporate, or guilt-tripping.

**Why:** The user explicitly cares about this. It's core to the app's identity — JustDilo is a personal companion, not a task manager.

**How to apply:** When writing or reviewing push message copy, AI prompts for push messages, or any reminder text — always ask: "would a friend actually say this?" Avoid generic phrases like "You have X tasks due." Favor specific, warm, energizing language that acknowledges the person, not just the task.
